# 代理开关

为了方便弄的插件，Chrome 和 Edge 都能用。

打包了 clash 后端，解压跑 setup.bat 就行，不用装别的东西。

## 安装

双击 setup.bat，然后浏览器加载扩展，完事。

以后开机自己跑，不用管。

## 三种模式

- 直连 — 不翻墙
- 规则 — 该翻的翻，国内直连
- 全局 — 全走代理

## 线路

内置 2 条线路，不通就换另一条。切换的时候自动拉最新配置，不怕 IP 过期。

## 注意

- 扩展加载需要开开发者模式，chrome://extensions 右上角
- 第一次用建议选全局模式测试能不能翻，通了再切回规则模式日常用
- 如果两条线都不通，等一会再试，上游可能换 IP 了
